# 🏊 Python Piscine Quests

Welcome to the Python Piscine! A "piscine" (French for swimming pool) is an intensive, self-paced programming boot camp style learning format. You are thrown into the "pool" and learn to swim by writing code.

This folder contains **100 practice questions** designed to build your skills from the ground up, starting from absolute beginner concepts to advanced asynchronous programming, structural pattern matching, and API design.

## 🛠 Rulebook: How to Solve

1. **Directories**: Each quest has its own folder (e.g., `q01_hello_world/`).
2. **Problem Statement**: Read the `README.md` inside the quest folder carefully. It contains:
   - Description of the task.
   - Input/output requirements.
   - Constraints (e.g., "Do not use built-in function `X`").
   - Hints.
3. **Write Your Solution**: Create your code inside the quest folder. Make sure your function or script meets the requirements.
4. **Test Your Code**: Run your script or write unit tests to ensure correctness.
5. **Review the Solution**: Each quest folder contains a `solution.py` file. **Only open it after you have attempted to solve the problem yourself!**
