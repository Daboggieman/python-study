# 📚 Python Crash Course (3rd Edition) — Book Info

## About the Book

**Python Crash Course (3rd Edition)** by Eric Matthes is one of the world's best-selling Python programming books. Published by No Starch Press, it covers Python fundamentals through hands-on projects.

### Part I: Basics (Chapters 1–11)
Covers variables, data types, lists, dictionaries, if statements, loops, functions, classes, files, exceptions, and testing.

### Part II: Projects (Chapters 12–20)
Three major projects:
- 🎮 **Alien Invasion** — A Space Invaders-style game (Pygame)
- 📊 **Data Visualization** — Charts and graphs (Matplotlib, Plotly)
- 🌐 **Web Applications** — A web app with Django
