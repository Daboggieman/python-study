# Enigma

Source: enigma

Files to submit
- solution.py

Allowed functions
- --allow-builtin

Instructions
Write a function that swaps values between several pointers in the requested order.

Expected function
```python
def enigma(a, b, c, d):
    pass
```


Here is a possible program to test your function :
```python
a = 1
b = 2
c = 3
d = 4
enigma(a, b, c, d)
print(a, b, c, d)
```

And its output :
```text
4 3 2 1
```
