# Lecture 02c: Operators and Expressions

Operators let you build expressions and perform computations in Python.

---

## 1. Arithmetic Operators

```python
print(10 + 3)
print(10 - 3)
print(10 * 3)
print(10 / 3)
print(10 // 3)
print(10 % 3)
print(2 ** 3)
```

- `+` addition
- `-` subtraction
- `*` multiplication
- `/` division
- `//` floor division
- `%` modulus
- `**` exponentiation

---

## 2. Comparison Operators

```python
print(5 > 3)
print(5 < 3)
print(5 == 3)
print(5 != 3)
```

These return `True` or `False`.

---

## 3. Logical Operators

```python
age = 20
is_student = True

print(age >= 18 and is_student)
print(age < 18 or is_student)
print(not is_student)
```

---

## 4. Assignment Operators

```python
x = 10
x += 2
x -= 1
x *= 3
```

These are shorthand for updating a variable.
