---

## 1. Project Planning
Before writing any code, it is essential to plan your project structure.
1. **Define Requirements**: What will the application do? Who will use it?
2. **Design Data Models**: How will data be stored? (Classes, dictionaries, JSON files).
3. **Map the Control Flow**: How will users interact with the program? (CLI menus, options).
4. **Determine Testing Strategy**: How will you verify that the functions work correctly?
