# Docker Level 7 Exercises

These expert-level exercises prepare you for real deployment workflows.

1. Write the Docker commands to tag an image as `myapp:1.0` and explain why tags matter.
2. Draft the commands for pushing an image to a registry and pulling it back.
3. Describe how you would use Docker in a CI/CD pipeline for building and testing an image.
4. Explain the difference between a production image and a development image.
