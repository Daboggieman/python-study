# Docker Level 3 Exercises

These exercises focus on file handling, ports, and inspecting containers.

1. Create a `Dockerfile` that copies a local folder into the image and prints its contents.
2. Run a container with a port mapping such as `-p 8080:80` and describe why it is needed.
3. Start a shell inside a running container with `docker exec -it <container> sh` and inspect the container file system.
4. Use `docker logs <container>` to view output from a stopped or running container.
