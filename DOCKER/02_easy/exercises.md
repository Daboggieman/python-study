# Docker Level 2 Exercises

Practice building and running a simple Docker image.

1. Create a `Dockerfile` for a Python script that prints "Docker is working".
2. Build the image with `docker build -t my-python-app .`.
3. Run the image with `docker run my-python-app`.
4. Use `docker images` and `docker ps -a` to confirm the image and container exist.
5. Remove the container and the image using Docker commands.
