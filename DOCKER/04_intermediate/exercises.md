# Docker Level 4 Exercises

These exercises introduce multi-container apps and Docker Compose.

1. Write a simple `docker-compose.yml` file that runs a web app and a database service.
2. Start the services with `docker compose up` and stop them with `docker compose down`.
3. Explain the difference between `docker run` and `docker compose up`.
4. Add a health check to one service and describe how Docker uses it.
