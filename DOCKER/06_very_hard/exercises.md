# Docker Level 6 Exercises

These exercises focus on security, configuration, and container health.

1. Modify a `Dockerfile` to run the application as a non-root user.
2. Add environment variables to the container using `ENV` in the Dockerfile or `-e` on `docker run`.
3. Add a `HEALTHCHECK` instruction to your Dockerfile and explain when it reports unhealthy.
4. Create a volume mount with `-v` and describe how it preserves data outside the container.
