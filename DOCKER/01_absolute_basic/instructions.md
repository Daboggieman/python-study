# Docker Level 1: Absolute Basic

Docker is a tool for running applications inside small, isolated containers.

- A container is like a tiny virtual computer that contains only the program and its needed files.
- Docker lets you run the same app on any machine without worrying about system differences.

### Key ideas:
- **Image**: a package with your app and its environment.
- **Container**: a running instance of an image.
- **Dockerfile**: a text file that tells Docker how to build an image.

### Example:
If you have a simple Python app, Docker can package Python and your code together so it works on another computer.

### Why it matters:
- You avoid "it works on my machine" problems.
- You can share your app with others easily.
