# Docker Level 5 Exercises

These exercises help you optimize images and understand build performance.

1. Create a `Dockerfile` that uses a multi-stage build to produce a smaller final image.
2. Add a `.dockerignore` file and explain what it prevents from being copied into the image.
3. Build the image and use `docker image ls` to compare the image size with and without optimization.
4. Describe why smaller images are better for production.
