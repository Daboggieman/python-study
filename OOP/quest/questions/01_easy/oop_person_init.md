Allowed functions

Instructions
Define a `Person` class with `__init__(self, name, age)` and a method `greet(self)` that returns the string `"Hi, I'm <name> and I'm <age> years old."`.

Expected function
```python
class Person:
    def __init__(self, name, age):
        pass

    def greet(self):
        pass
```


Here is a possible program to test your function :
```python
p = Person("Ada", 30)
print(p.greet())
```

And its output :
```text
Hi, I'm Ada and I'm 30 years old.
```
