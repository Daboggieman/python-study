# Event Scheduler

Write a function named `has_overlap(events)` that checks whether any two time ranges overlap.

## Requirements
- Each event is a tuple of `(start, end)`.
- Assume all times are integers and that the end time is greater than the start time.
- Return `True` if any overlap exists, otherwise `False`.

## Example
```python
has_overlap([(1, 3), (2, 5), (6, 8)])
# -> True
```
