# 📊 DSA — Data Structures & Algorithms

A dedicated module covering the data structures and algorithms every developer eventually needs, built in the same two-part style as the rest of this repo:

- **`curriculum/`** — 19 lessons (`lecture.md` + `exercises.py`), same format as the root `curriculum/` folder. Read the lecture, then fill in the TODOs in the exercises file.
- **`quest/`** — 38 practice problems in the same style as the root `python-quest/` folder, spread across four difficulty tiers (`01_easy` → `04_extremely_hard`), each with a question spec (`quest/questions/<tier>/`) and a starter answer stub (`quest/answers/`).
- **`resources/`** — `cheatsheet.md` (one-page reference for everything in this module) and `links.md` (every W3Schools/YouTube/article link used throughout, consolidated).

## Suggested Study Flow

1. Read `curriculum/NN_topic/lecture.md`.
2. Complete `curriculum/NN_topic/exercises.py`.
3. Solve the matching `quest/` problems for that topic (check `quest/questions/*/` — filenames are prefixed with the topic name).
4. Keep `resources/cheatsheet.md` open as a running reference while you work.

This module assumes you're comfortable with the fundamentals in the root `curriculum/` folder (functions, classes, loops) — start there first if you haven't already.
